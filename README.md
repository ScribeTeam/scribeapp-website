# Scribe App Website

To start the development server:
```
yarn dev:website
```

Once development is done, we can build the website using:
```
yarn build:website
```

Commit all the chagnes, first push them to origitnal repo and than we can push to heroku for updating the live server.

```
git push heroku master
```


# Bugs

If unable to launch, check node package is up to date & the correct node version
```
node -v
```
